# 更新日志

## v0.1.0.0 (2026-07-04)

v0.1.0.0 诞生！至此主要开发已完成。

### 新增
- Toast 弹窗通知系统（右上角滑入 → 停留 → 上滑消失，多弹窗堆叠）
- JSON 导入结构校验（验证 `particleParams` 字段）

### 变更
- 替换所有 `alert()` 为统一 Toast 弹窗
- render.js / sequence.js 错误改为仅弹窗提醒，面板只保留状态显示
- 加载成功 / 导入成功改为 Toast 通知
- 默认时间戳输入框改为空
- 去除所有 emoji（状态栏和按钮）
- 工具栏/控制面板/播放条降低背景透明度，毛玻璃效果真正可见
- README 添加 Netlify 在线体验链接

### 修复
- Toast 动画改用 `@keyframes`，修复原 CSS transition 不触发的问题
- JSON 导入前校验，无效文件不再导致 `restoreState` 异常
